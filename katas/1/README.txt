En este problema recibiremos dos array para encontrar la posicion del item
igual en el segundo array. Para resolver este problema se utilizara un
recorrido convencional por medio de un for en el que se estara iterenado
cada item, dentro del for estara un if que basicamente va a revisar que la
condicion de compla de A[i]== B[j] si esto resukta verdadero se va a guardar
o imprimir en consola la posiscion de dicho elemento. Todo esto se va a
ejecutar hasta recorrer todo el array.
