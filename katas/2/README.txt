Caso I:
El primer paso que se revisa en este problema es que la longitud sea la adecuada
una vez que esto se cumple se pasa al caso II.

Caso II:
Se verifica caracter por caracter que se cumplan las condiciones de contener
un digito, una mayuscula y una minucula, si las condiciones se cumplen se ponen
en true unas banderas que nos ayudaran a identificar si la contrasena es aceptada
o no.

Caso III:
La contrasena no es aceptada. Entonces se entra a una serie de if que nos van
a ayudar a identificar el porque no es aceptada y le arrojaran al usuario
el mensaje que error con una breve descripcion.
